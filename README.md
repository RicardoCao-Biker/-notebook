### 基于Vue的云笔记APP
一个基于Vue的云笔记APP，功能向印象笔记看齐，是在饥人谷里的Vue项目，正在开发中...

使用Vue全家桶，实现注册登录功能，新增笔记本，在笔记本中添加笔记、删除，回收站等功能。
